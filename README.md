# Travel Booking Website UI

## Overview
This project is a front-end web application that simulates a travel booking platform. It provides a multi-page user interface for booking flights, trains, and hotels. The design focuses on usability, responsiveness, and clean navigation.

## Features
- Multi-page layout (Home, Airplane, Train, Hotel, Profile)
- Responsive navigation bar
- Image slider for showcasing travel options
- User authentication pages (Sign Up / Sign In)
- Clean and structured UI design

## Tech Stack
- HTML5
- CSS3
- JavaScript
- jQuery
